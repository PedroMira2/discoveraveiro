Este repositório contém a versão estática do site Discover Aveiro, originalmente desenvolvido em WordPress.
A conversão foi realizada utilizando a ferramenta HTTrack, que permite clonar sites e gerar seus arquivos HTML, CSS, JS e assets.

Processo de Conversão
A migração para um site estático seguiu estes passos:

Utilização do HTTrack para capturar a estrutura completa do site WordPress.

Ajustes manuais no código gerado, uma vez que a conversão automática não é perfeita.

Correções aplicadas:

Substituição ou recuperação de imagens que não foram corretamente exportadas.

Ajustes de fontes e estilos que ficaram desformatados após a clonagem.

Pequenas melhorias no HTML/CSS para garantir consistência visual e funcional.

Objetivo
O objetivo desta conversão é disponibilizar uma versão estática, leve e rápida do site, ideal para hospedagem simples, maior segurança e menor necessidade de manutenção.
